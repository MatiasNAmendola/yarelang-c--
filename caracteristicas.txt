-- Variables globales --
yare cuenta con 26 variables globales, de la 'a' a la z.

Las cuales pueden ser usadas así:

<codigo>
a = 123;
printn(a);
.
</codigo>

ó así:

<codigo>
printn(A);
</codigo>

En las variables globales el caso no importa.

-- Comentarios --

En Yare hay distintos tipos de comentarios, almenos 5 tipos de comentarios:

Comentarios tipo shell script, python o perl:

<codigo>
a = -12; # Soy un comentario ... 
</codigo>

Comentarios tipo C/C++/Java,etc (multilínea)

<codigo>
a = -12; /* 
	Soy 
	un 
	comentario 
	*/
</codigo>

Comentarios tipo basic con "rem":

<codigo>
rem Soy un comentario
</codigo>

Comentarios estilo Batch:

a = -12; :: Soy un comentario

Comentarios estilo SQL:

a = -12; -- Soy un comentario
